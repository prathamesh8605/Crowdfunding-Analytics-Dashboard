# Crowdfunding-Analytics-Dashboard
Excel-based Crowdfunding Analytics Dashboard analyzing project success rate, goal ranges, categories, and funding trends using data visualization techniques.
📌 Project Overview

This project focuses on analyzing crowdfunding campaign data to identify key factors affecting project success. The dashboard provides insights into successful projects, funding trends, goal ranges, categories, and backer behavior.

🎯 Objectives

Calculate percentage of successful projects
Analyze success rate by category
Analyze success rate by goal range
Identify top funded successful projects
Study number of backers and pledged amount trends

🛠 Tools Used

Microsoft Excel
Pivot Tables
Power Query
Data Cleaning
Data Visualization
Basic Data Analysis Concepts
Power Bi 
Dax
Tablleau
MySql

📈 Key Insights

Identified overall success percentage of projects
Found which category has highest success rate
Analyzed goal ranges that have higher success probability
Determined top funded successful projects
Studied backers contribution trends

📊 Dashboard Features

KPI Cards (Total Projects, Successful %, Total Backers, Total Pledged)
Category-wise Success Analysis
Year & Month Trend Analysis
Goal Range Success Analysis
Top Funded Projects

📁 Dataset

The dataset contains crowdfunding campaign details including:

Project Name
Category
Goal Amount (USD)
Pledged Amount (USD)
Backers
Outcome
Launch Date

🚀 Conclusion

The analysis shows that moderate funding goals have higher success rates, and certain categories consistently perform better. Backer engagement plays a crucial role in project success.
